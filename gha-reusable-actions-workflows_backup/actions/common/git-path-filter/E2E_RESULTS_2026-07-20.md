## Bugs found & fixed during e2e

1. Passing `base=<sha>^` into `git fetch` / resolve failed.
2. `git fetch --depth=N` re-shallowed full checkouts and dropped parents.
3. Auto-detect on `workflow_dispatch`+`main` compared `main` vs `main` (empty).

## How to reproduce

```bash
gh workflow run git-path-filter-e2e.yml --ref main
gh run watch --exit-status
```
