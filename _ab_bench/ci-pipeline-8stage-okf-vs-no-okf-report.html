11. Methodology

  • Same model; parallel OKF vs no-OKF subagents; parent never implemented.
  • Shared functional brief only — no SPVS/Conftest/SHA rules in baseline initial prompt.
  • OKF: okf.py lookup --card --limit 3; Prompt Cards only. Baseline: no Aegis/policy trees.
  • Parent Conftest FAIL → resume arm with raw failure (max 2 loops).
  • Scored: true wall (sum of runs) + parent runs to PASS + tokens/cost/quality.
  • Not scored alone: draft wall, tool-call turn counts, tok/s throughput.
